Description: 
This project is a grid based inventory management system similar to what is found in Escape from Tarkov.

Author: Quinn O'Connor

The requirements are:

* CMake 3.0 or better
* A C++17 compatible compiler
* Git
* Doxygen (optional)

To configure:

```bash
cmake -S . -B build
```

Add `-GNinja` if you have Ninja.

To build:

```bash
cmake --build build
```
